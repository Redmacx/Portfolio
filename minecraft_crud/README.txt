🖥️ How to Set Up the Project:

✅ 1. Set Up the Database
1. Open XAMPP and start Apache and MySQL.
2. Go to your browser and open phpMyAdmin: http://localhost/phpmyadmin
3. Click on "Import" tab.
4. Choose the file: bookdb.sql
5. Click Go to import the database.

✅ 2. Open the Project in vscode
1. Open vscode 
2. Go to File > Open Project
3. Browse to the folder /Minecraft theme (CRUD)/ and open it.
4. Wait for vscode to load all files and dependencies.

✅ 3. Run the Program
1. Right-click the project in NetBeans
2. Click Run
3. The system should open with Minecraft-themed CRUD interface 🎮
